Welcome to the beginner Python curriculum.

## What is Python?
A beginner-friendly programming language used in apps, AI, automation, games, and more.

## Why Learn Python?
- Great for beginners
- Easy to read and write
- Used by real companies (Youtube, ChatGPT, Spotify)

## Will Coding be Replaced by AI?
- AI won’t replace Python just like calculators didn’t replace math
- You still need people to think, understand, and design the code
- AI is a tool that makes Python faster and easier, not unnecessary
