# Unit 2 — Variables & Data Types

## What Are Variables?
Values your game can store and change (name, health, score, weapon).

## Why Are They Important?
- Track player stats
- Store choices
- Update game state
- Keep score and inventory

## Data Types
- String → text ("hello world", "aaron", "Jack")
- Integer → whole numbers (1, 68 , 100)
- Float → decimals (1.0, 2.4, 0.0, 3.14)
- Boolean → True/False

## Making a Variable
 - Variable name = value
 - Ex. age = 24

## Snake Case
 - If a variable has 2 or more words, use an underscore and lowercase letters
 - Ex. is_on = False
